# Civil Engineering Insight Studio

## Project Description
AI powered application that analyzes civil engineering images using Google Gemini Pro Vision API and provides engineering insights.

## Features
- Image based structure analysis  
- Material identification  
- Defect detection  
- Safety recommendations  
- AI powered explanation  

## Technologies
- Python  
- Flask  
- Google Gemini Pro  
- HTML  

## How to Run

1. Install libraries  
pip install -r requirements.txt

2. Add API key in .env

3. Run  
python app.py

4. Open  
http://127.0.0.1:5000
